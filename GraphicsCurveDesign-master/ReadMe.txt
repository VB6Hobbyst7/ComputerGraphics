


1. Open the graphics_hw3 folder
 
	cd graphics_hw3 


2. Build the project using the makefile
 
	make all


3. Run the program
 
	./main


Note: 

CONTROL POINTS INPUT: Control Points can be entered with Left Mouse Button Click in OpenGL Window

SUBDIVISION: Through UP/DOWN arrow key, Sub Division Level can increased/decreased.

BONUS : Control Points for the surface patch are initialized beforehand.
